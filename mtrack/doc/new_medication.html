Adding medications to the formulary

To create a new entry in the formulary go to the Formulary tab. Click on the New Medication button.

From there you must enter in the details for the new medication. The Generic for and Unit size fields are optional.

You may (optionally) add default instructions. This will automatically place a template for the prescription instructions every time you select this particular medication. For example every time you prescribe the new medication the usage instructions may only vary by dose size and duration. Click Finish when you are done.

You may now start adding inventory for the new medication.

Modifying an existing medication

To modify an existing medication go to the Formulary tab. Search for the medication you wish to modify and right-click the entry. Select Modify Medication. From here on the process is the same as adding a new medication.
