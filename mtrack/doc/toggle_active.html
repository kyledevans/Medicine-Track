Making things active or inactive

You can make patients, pharmacists, prescribers, medication, or specific inventory active or inactive. For each the results are:

  • Inactivating Patients - Filters patients from appearing in patient searches.
  • Inactivating Medications - Prevents new inventory from being created as well as new prescriptions from using the medication.
  • Inactivating Pharmacists - Prevents new prescriptions from being filled by the pharmacist.
  • Inactivating Prescribers - Prevent new prescriptions from being written by the prescriber.
  • Inactivating Inventory - Prevents new prescriptions from using the medicine. Also no longer includes any remaining inventory in the Formulary.

    • Note that the active / inactive state of any of these should never effect the Prescription log. If you find a case where it does. Make note of it so that it can be fixed.

    For all cases in order to mark an item as invalid first search for it in the relevant tab. Right click the entry and select Toggle active. If you need to re-activate an item then you will need to uncheck Active when searching in order to have inactive items displayed.

    Marking prescriptions as invalid

    Only mark a prescription as invalid when the prescription entry has an error or the patient never received the medication. If a prescription entry needs to be invalidated because of an error, then a new prescription entry needs to be created with the correct information.

    To invalidate a prescription, search for it in the Prescription log tab. Right-click the entry and select Invalidate. Note that you cannot re-validate a prescription once it has become invalid.
