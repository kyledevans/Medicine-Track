Creating a new prescription

To create a new prescription go to the Patients tab. Search for and then select the patient you need. Then either right click the patient on the list and select New Prescription or click the button above the list.

Select a medication that you wish to prescribe. You can either scan a barcode or search for a medication by name. The example image below is using a barcode reader.

The next screen allows you to fill in information about the prescription. Currently it is possible to enter an amount greater than the current inventory. If this happens the wizard will give an error once you press Finish and the prescription will not be recorded. If your prescription needs to take inventory from 2 sources, see the relevant entry underneath "See Also" below.

Enter the instructions for the patient.

Once you have finished the prescription a print dialog will pop up and allow you to print a label.
