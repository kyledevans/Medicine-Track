Creating a new prescription

To create a new prescription go to the Patients tab. Search for and then select the patient you need. Then either right click the patient on the list and select New Prescription or click the button above the list.

Select a medication that you wish to prescribe. You can either scan a barcode or search for a medication by name. The example image below is using a barcode reader.

The next screen allows you to fill in information about the prescription. Currently it is possible to enter an amount greater than the current inventory. If this happens the wizard will give an error once you press Finish and the prescription will not be recorded. If your prescription needs to take inventory from 2 sources, see the relevant section below.

Enter the instructions for the patient.

Once you have finished the prescription a print dialog will pop up and allow you to print a label.

Prescribing from multiple sources

Here is how to prescribe from multiple sources. For example if there are 2 bottles of Penicillin tablets. Bottle A has 4 more tablets left while bottle B has 100. If you want to prescribe 20 tablets this is what you should do:

  1. Make a prescription for 4 tablets from bottle A.
  2. Make a prescription for 16 tablets from bottle B.
  3. Print a label for either the first or second prescription.
  4. On the label where it says how many were dispensed, with a pen cross out what is there and write in the correct amount.
