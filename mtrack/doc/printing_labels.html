Printing labels

Printing prescription labels

Normally a label for a prescription will be printed immediately after writing a new prescription. However if you need to print a label after the fact go to the Prescription log tab. Search for and select the prescription that you need to print.

Right-click the prescription entry and select Print Label. Select the DYMO Right printer and click print.

Printing inventory barcodes

Normally inventory labels will be printed immediately after adding them to the system. However if you need to print a label after the fact go to the Inventory tab. Search for and select the inventory entry you need.

Right-click the inventory entry and select Print Barcode. Select the DYMO Left printer and click print.
