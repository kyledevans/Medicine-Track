Adding a new pharmacist

To create a new pharmacist go to the Pharmacists tab. Click on the New Pharmacist button.

Fill in the last and first name. The Initials will appear on the prescription bottle labels.

Modifying an existing pharmacist

To modify an existing pharmacist go to the Pharmacists tab. Search for the pharmacist you wish to modify then right-click the entry. Select Modifu Pharmacist. From here on the process is the same as adding a new pharmacist.
